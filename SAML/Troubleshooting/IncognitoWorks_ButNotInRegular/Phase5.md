# Phase 5 -- Collect Customer Environment Data

## Objective

Gather environment-specific factors affecting authentication.

------------------------------------------------------------------------

## Step 1 -- Provide System Details

-   OS version
-   Browser version
-   Domain-joined status

------------------------------------------------------------------------

## Step 2 -- Provide Network Details

-   VPN usage
-   Proxy configuration

------------------------------------------------------------------------

## Step 3 -- Provide IdP Logs

-   Azure AD sign-in logs
-   Authentication method used

------------------------------------------------------------------------

## Step 4 -- Provide Browser Config

-   Trusted sites
-   WIA settings

------------------------------------------------------------------------

## Deliverables

All collected environment data.

------------------------------------------------------------------------

## Expected Outcome

Identify environmental differences causing the issue.
